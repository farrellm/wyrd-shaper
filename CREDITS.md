# Credits

redistributed, so supply them locally):

- **UI asset pack** by [Franuka](https://franuka.itch.io/) — the
  `FantasyRPGtext` and `FantasyRPGtitle` fonts used for in-game UI text
  (`assets/ui_pack/`). Used under the pack's terms: free for commercial and
  non-commercial projects with a link back.
- **Fantasy RPG asset pack** by [Franuka](https://franuka.itch.io/) —
  overworld terrain art: grass, coastlines, stone roads, forest, rocks,
  flowers, cave entrance (`assets/asset_pack/`). Same terms as above.
- **Fantasy RPG — Desert pack** by [Franuka](https://franuka.itch.io/) —
  the scrub biome's sand tiles (`assets/desert_pack/`). Same terms.
- **Fantasy RPG — Castles pack** by [Franuka](https://franuka.itch.io/) —
  the start area's brick walls (`assets/castles_pack/`). Same terms.
- **Fantasy RPG — Elemental Dungeons (Fire) pack** by
  [Franuka](https://franuka.itch.io/) — dungeon floors, walls, stairs, the
  door, the summoning circle, and torches
  (`assets/dungeons_fire_pack/`). Same terms.
